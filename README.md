## 🚀 Despliegue y Desarrollo

### Ejecutar el servidor de desarrollo
```bash
npm install
npm run dev
```

### Build para producción
```bash
npm run build
```

### Deploy
El proyecto está configurado para **deploy estático** en Render.com. Simplemente conecta tu repositorio de GitHub y Render detectará los cambios automáticamente.
